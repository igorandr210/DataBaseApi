# DataBaseApi

Api methods provide:

* Get all stored data;
* Get item by ID;
* Insert new item;
* Delete item;
* Update item; 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

## Running the solution

use 'docker-compose up' to start up project
